```text
V4   = conceptual frame
V4.1 = expanded framework and thought-experiment architecture
V4.2 = mathematical toy-model development and current active baseline
```

```text
V4 introduced the conceptual frame of the 6D Projection Framework.

V4.1 expanded and clarified that frame, developing the 3S+1T / 3T+1S interpretation and the main thought-experiment architecture.

```

Therefore:

```text
V4 and V4.1 are historical foundation layers.
```

## V4 / V4.1 status

Older V4 / V4.1 files may still exist in the repository history or in legacy folders.

They are retained as historical development material.

They should not be treated as the current public baseline.

If this package is being uploaded into an existing V4.1 repository, the older V4 / V4.1 materials should be moved into a clearly marked folder such as:

```text
legacy_v4_v4.1/
```

or clearly labelled as historical in the repository description.
