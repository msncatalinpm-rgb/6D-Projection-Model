# License Decision Required

No final license is selected in this package.

Before publishing publicly, decide:

## For documentation

Possible options:

```text
CC BY 4.0       allows reuse with attribution
CC BY-NC 4.0    allows reuse with attribution, non-commercial only
All rights reserved / no license
```

## For code

Possible options:

```text
MIT
Apache-2.0
GPL-3.0
```

## Recommendation

For first release, keep the repository public only after the license decision is made.

A common split is:

```text
documents: CC BY 4.0 or CC BY-NC 4.0
code: MIT or Apache-2.0
```

This is not legal advice.
