# Reviewer Orientation - Toy Model 5G Public Baseline

This package should be read from the public front door first: the root `README.md`.

The root README now explains:

1. the model category: integrated geometric research architecture;
2. the Embedded Observer Problem and accessibility-boundary interpretation;
3. the Toy Models 1-5 and the geometric interpretation each tested;
4. why Toy Model 5G was needed;
5. what the full-SPARC 5G-M run found;
