# Data Inventory

## Current Data Status

This repository contains Toy Model output summaries and documentation.

It does not yet contain a clean reproducible source-code/data pipeline.

## Included output-data types

```text
Numerical summary TXT files
Rebind summary TXT files
Plots embedded in PDFs and, for Toy Model 1, some PNG development figures
```

## Not included yet

```text
raw observational datasets
SPARC galaxy tables
standalone scripts for every toy model
Jupyter notebooks
requirements.txt / environment.yml
unit tests
```
