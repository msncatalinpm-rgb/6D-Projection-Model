## Recommended release decision

Publish the V4.2 foundation before incorporating Toy Model 5G.

Reason:

```text
The framework baseline is closed enough to share.
5G is real-data work and will introduce dataset, parameter, and fitting risks.
Those should be handled openly as post-release work, not mixed into the foundation release.
```
