# Contributing

This project welcomes critical review.

Good contributions include:

```text
mathematical corrections
failed replications
cleaner derivations
clearer guardrails
data-protocol improvements
code reproducibility improvements
```

Bad contributions include:

```text
overclaiming
unfalsifiable assertions
adding parameters only to force a fit
presenting toy models as proven physics
```

## Review Principle

Failure is useful.

If a test fails, preserve the failure and document what broke.
